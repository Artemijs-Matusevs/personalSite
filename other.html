SUDOKU

  • Command line application of the game "Sudoku" written in Java.
  • To compile and run (requires java to be installed on the machine):
    • Open cmd and change directory to directory of the source code using cd command.
    • Type "javac sudoku2.java" - This will compile the program.
    • Run the program by typing "java sudoku2".
