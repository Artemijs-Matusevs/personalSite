Pathfinding AI

  • Download and extract zip, run "EnemyAI.exe"
  • The capsule will find the shortest path to the red cube, move the cube by moving the mouse.
  • There are 3 states:
    • Patrol: A random coordinate within a set area from the capsule gets generated, using a raycast I was able to validate that the generated coordinate is on a walkable surface. Once this "walkPoint" gets reached, a new coordinate gets generated.
    • Chase: When the red cube gets within a certain range from the capsule, the capsule will find the fastest path to the cube.
    • Attack: Once the capsule gets within a certain range from the cube, it will stop chasing and stand in one place. The attack method is currently empty
  • The pathfinding function was achieved with the use of Unity's built in NavMesh mechanic.

Gun Prototype

  • Download and extract zip, run "shootAnimationPrototype1911.exe"
  • To Move (W,A,S,D). Reload (R). View ammo (T). Reset level (F).
  • All animations have been imported from Blender and are set up within the Unity animator engine using the different states.
    • Shoot
    • Shoot Empty
    • Reload
    • Reload Empty
    • Walking Key Down
    • Walking Empty
  • Using raycasts, I can identify the objects which are hit, I was able to add force to the cone at the point of impact.
  • There are two camera objects set up rendering the scene. One renders everything but the gun. The other only renders the gun. Using this method I was able to avoid the gun clipping through any walls.
